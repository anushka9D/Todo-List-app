# Todo-List-app

## Setup Instructions

### Prerequisites
- Node.js v16+ and npm install

### Steps
1. **Clone the repository**:
   ```bash
   git clone URL
   cd Todo-List-app/backend
   npm start
   ```

   ```bash
   cd Todo-List-app/frontend
   npm start
   ```

**Open [http://localhost:3000](http://localhost:3000/login) in your browser.**
